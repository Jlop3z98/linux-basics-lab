# linux-basics-lab
Beginner Linux lab documenting basic command line skills and setup.

## Objective
Learn basic Linux commands and navigation
 
 ## Commands Learned
 - pwd: shows current directory 
 - ls: lists files and folders
 - cd: change directory
 - mkdir: create a new folder
 - touch: create a file

   ## What I Did
   - Installed Ubuntu on a virtual machine
   - Practiced navigating directories
   - Created folders and files using terminal

   ## Takeaways
   - Learned how to move around the Linux file system
   - Understood basic command line usage
